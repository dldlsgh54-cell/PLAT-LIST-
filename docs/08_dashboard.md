# Dashboard

The Dashboard shows what the user needs to work on today.

It is the daily operations view for Creator OS, focused on production status, blockers, waiting queues, and approval items.

## Display Items

- Today's task list
- Relaxing production progress
- Vocal production progress
- Shorts waiting items
- Render queue
- Upload queue
- Gemini connection status
- Missing files
- Error projects
- Projects under the 60-minute requirement
- Items waiting for user approval

## Dashboard Priorities

- Surface blocked projects clearly.
- Show user approval items before automated work continues.
- Make missing files and render blockers visible.
- Highlight vocal projects that do not meet the 60-minute minimum.
- Keep long-form and shorts production status connected.
