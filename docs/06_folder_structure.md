# Folder Structure

Creator OS uses the following managed workspace structure.

```text
CreatorOS/
|-- Projects/
|   |-- Rest/
|   |   |-- Somnera/
|   |   `-- Noctis_Atlas/
|   `-- Vocal/
|       |-- Aurora_Echo/
|       `-- Velvet_Verse/
|-- Library/
|   |-- Ambient/
|   |-- Images/
|   |-- Videos/
|   |-- Audio/
|   |-- Lyrics/
|   |-- Fonts/
|   |-- Logos/
|   |-- Prompts/
|   |-- Player_Themes/
|   `-- Templates/
|-- Settings/
`-- Logs/
```

## Top-Level Areas

- `Projects`: Channel-specific production projects.
- `Library`: Shared source assets and reusable production resources.
- `Settings`: Application configuration files.
- `Logs`: Runtime logs and diagnostic output.

## Project Areas

- `Projects/Rest`: Relaxing channel projects.
- `Projects/Vocal`: Vocal channel projects.

## Library Areas

- `Ambient`: Ambient audio and background sound assets.
- `Images`: Still image assets.
- `Videos`: Video assets and source loops.
- `Audio`: Music and audio files.
- `Lyrics`: Lyrics and subtitle source text.
- `Fonts`: Font files used in renders.
- `Logos`: Channel logos and branding assets.
- `Prompts`: AI prompts for Gemini and related workflows.
- `Player_Themes`: Visual player themes and layout presets.
- `Templates`: Reusable render, subtitle, and project templates.
