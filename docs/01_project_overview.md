# Creator OS

## Purpose

Creator OS is a Python desktop application for operating an AI music channel.

It is not a simple video editor. It is an operating system for managing:

- Relaxing long-form videos
- Vocal playlist videos
- Shorts production

## Core Goals

- Music file management
- Automatic A/B audio classification
- Gemini evaluation
- Python audio analysis
- 60-minute playlist validation
- Subtitle generation
- Automatic video rendering
- Relaxing loop video production
- Derived shorts production
