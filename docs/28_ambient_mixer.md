# Ambient Mixer

Ambient Mixer provides automatic mixing for music and ambient layers.

## Example Mix

```text
Music 40%
Rain 45%
Room Tone 10%
Traffic 5%
```

## Features

- Volume sliders
- Automatic loop
- Crossfade
- Fade in / out
- Random Mode
- Prefer selected files
- Save Profile

## Rules

- Automatic Mix should follow the selected Room Profile or Journey Profile.
- Users can adjust layer volume with sliders.
- Crossfade should prevent obvious loop cuts.
- Fade in and fade out should be available for final render polish.
- Random Mode should only choose files from allowed profile categories.
- Preferred files should be selected before other candidates when available.
- Saved Profiles can be reused in future projects.
