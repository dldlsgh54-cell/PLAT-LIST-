# Vocal Studio MVP

The first MVP of Creator OS is the vocal production workflow.

## MVP Goal

Build the minimum usable workflow for creating and validating vocal playlist projects.

## Features

1. Create project
2. Select channel
3. Set default target duration to 60 minutes
4. Upload Excel lyrics
5. Generate Suno work track list
6. Drag and drop import for A/B audio files
7. Automatic A/B matching
8. Python audio analysis
9. Gemini A/B evaluation
10. Automatic winner selection
11. Calculate total playlist duration
12. Block rendering when under 60 minutes
13. Show missing duration
14. Notify when additional tracks are needed

## MVP Rules

- Vocal projects default to a 60-minute target duration.
- Rendering is blocked until the selected winners reach at least 60 minutes.
- A/B matching should reduce manual sorting work.
- Winner selection should be automatic but still reviewable by the user.
- Python analysis and Gemini evaluation both contribute to A/B decisions.
- The project should clearly show how much additional duration is needed.
