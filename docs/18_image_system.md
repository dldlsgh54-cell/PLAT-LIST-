# Image System

Creator OS uses different image workflows for relaxing projects and vocal projects.

## Relaxing Projects

Required assets:

- One still image
- One flow loop video

## Vocal Projects

Required assets:

- One image per track
- A 20-track project usually needs 20 images
- Visual consistency is maintained through a Visual Bible

## Vocal Image Workflow

1. Generate Visual Bible
2. Extract the core emotion of each track
3. Generate track-specific image prompts
4. Generate and register images
5. Gemini evaluates the match between each song and image

## Rules

- Vocal project images should feel visually unified across the playlist.
- Track images should still reflect the emotion and identity of each song.
- The Visual Bible defines shared art direction, palette, mood, character rules, framing, and brand fit.
- Gemini image matching is advisory, not final approval.
- Missing track images should be shown as project blockers.
