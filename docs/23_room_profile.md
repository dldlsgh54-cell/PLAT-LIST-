# Room Profile

Somnera uses the Room Profile system.

One video equals one room and one theme.

## Room Examples

### Rain Room

- Rain
- Room Tone
- Optional Light Traffic

### Fireplace Room

- Fireplace
- Room Tone
- Optional Wood Ambience

### Ocean Room

- Ocean Waves
- Sea Breeze
- Room Tone

### Snow Room

- Soft Wind
- Snow Atmosphere
- Room Tone

## Room Lock

Room Lock prevents ambient categories outside the selected Room Profile.

## Rules

- A Somnera video should stay inside one room concept.
- Ambient assets must match the selected Room Profile.
- Optional ambient categories are allowed only when listed in the profile.
- Unapproved ambient categories should be blocked or flagged.
- Room Profile selection controls the default ambient recommendations.
