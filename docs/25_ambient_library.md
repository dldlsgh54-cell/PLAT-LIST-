# Ambient Library

Creator OS does not include built-in ambient sound files.

Because of licensing risk, the user provides ambient files that are free and allowed for commercial use.

## Recommended Sources

- Pixabay Sound Effects
- Mixkit Sound Effects

## User Workflow

1. User downloads ambient sound files from a license-safe source.
2. User places the files in `Library/Ambient`.
3. Creator OS scans the ambient library.
4. Creator OS analyzes and classifies the files by listening to the audio.

## Rules

- Creator OS should not ship with ambient sound assets.
- File names do not need to follow a fixed naming convention.
- Classification should be based on audio analysis, not only file names.
- The user is responsible for confirming license suitability.
- Ambient files can be used by Room Profiles and Journey Profiles after classification.
