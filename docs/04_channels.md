# Channels

Creator OS starts with four default channels.

## Relaxing Channels

### Somnera

- Type: Relaxing
- Concept base: Room
- Focus: Sleep, rest, room concepts
- Example concepts:
  - Rain Room
  - Fireplace Room
  - Library Room

### Noctis Atlas

- Type: Relaxing
- Concept base: Journey
- Focus: Night travel and moving spaces
- Example concepts:
  - Train
  - Ocean Liner
  - Airship
  - Highway

## Vocal Channels

### Aurora Echo

- Type: Vocal
- Language: English
- Vocal style: Female vocal
- Genres:
  - Pop
  - Dream Pop
  - Indie Pop

### Velvet Verse

- Type: Vocal
- Language: English
- Vocal style: English vocal
- Genres:
  - Cinematic Pop
  - Emotional Pop
