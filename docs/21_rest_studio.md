# Rest Studio

Rest Studio manages the relaxing long-form production workflow.

## Workflow

1. Create project
2. Select channel
3. Select Room Profile or Journey Profile
4. Register music files
5. Automatically apply Ambient Profile
6. Register image
7. Register flow loop video
8. Create Master Playlist
9. Select target video duration
10. Automatically calculate loop count
11. Render Preview
12. Final render

## Channel Profile Direction

- Somnera uses Room Profiles.
- Noctis Atlas uses Journey Profiles.

## Rules

- Relaxing projects use a 50-70 minute master playlist.
- The master playlist is looped to reach the selected target video duration.
- The Ambient Profile should match the selected Room or Journey Profile.
- One still image and one flow loop video are required.
- Render Preview should be available before final render.
