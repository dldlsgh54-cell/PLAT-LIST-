# Visual Bible

Vocal albums use a Visual Bible to keep images consistent across the project.

The Visual Bible defines the album's shared visual world.

## Elements

- Overall color palette
- Lighting
- Background world
- Character mood
- Clothing
- Art style
- Lens feeling
- Album mood
- Forbidden elements

## Rules

- Track images should be different scenes within the same album world.
- Images should not feel like unrelated standalone artworks.
- Shared visual rules should remain consistent across all track images.
- Track-specific emotion can vary, but the album identity must remain clear.
- Forbidden elements should be checked before image approval.

## Purpose

- Maintain album-level visual consistency.
- Support track-specific emotional scenes.
- Make vocal playlists feel like one cohesive project.
- Reduce visual mismatch between songs and images.
