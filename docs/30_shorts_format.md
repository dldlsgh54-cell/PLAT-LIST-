# Shorts Format

Creator OS uses a standard vertical format for Shorts.

## Default Format

- 9:16 vertical video
- Center album image
- Audio spectrum
- Lyric subtitles
- Song title
- Channel name
- Bottom full version notice
- Long-form link in pinned comment

## Duration Candidates

- 20 seconds
- 30 seconds
- 45 seconds
- 60 seconds

## Rules

- Shorts must be rendered vertically.
- Shorts should clearly identify the song and channel.
- The source long-form video should be linked through metadata and upload data.
- The bottom full version notice should direct viewers to the related long-form video.
- Subtitle layout should respect mobile safe area rules.
