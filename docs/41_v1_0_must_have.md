# v1.0 Must Have

Creator OS v1.0 focuses on the vocal production workflow and the minimum application foundation required to render usable long-form playlist videos.

## Required Features

1. PySide6 GUI
2. Dashboard
3. Vocal production window
4. Project creation
5. Excel lyrics upload
6. A/B audio drag and drop import
7. Automatic A/B matching
8. Python audio analysis
9. Gemini connection check
10. Gemini A/B evaluation
11. Automatic winner selection
12. 60-minute Playlist Manager
13. SRT subtitle generation
14. FFmpeg Preview Render
15. FFmpeg Full Render
16. Preflight Check
17. Non-destructive file handling
18. Log storage
19. Settings

## v1.0 Scope Rules

- v1.0 prioritizes the vocal workflow before the full relaxing and shorts workflows.
- Rendering must be blocked when Preflight Check fails.
- Vocal playlist rendering must be blocked under 60 minutes.
- Preview Render must be available before Full Render.
- Source files must not be modified.
- Gemini integration must be health-checked and visible.
