# App Structure

Creator OS is a single desktop program with the following primary menu sections.

## Main Menu

1. Dashboard
2. Relaxing Production
3. Vocal Production
4. Shorts Production
5. Library
6. Analytics
7. Settings

## Menu Purpose

### Dashboard

Central overview for channel production status, recent files, pending checks, and render progress.

### Relaxing Production

Workflow for long-form relaxing music videos, including loop video creation and 60-minute playlist validation.

### Vocal Production

Workflow for vocal playlist videos, including track selection, subtitles, lyrics sync, and playlist rendering.

### Shorts Production

Workflow for creating short-form derivative videos from existing music, playlists, or rendered long-form content.

### Library

Music, image, video, subtitle, and project asset management.

### Analytics

Review of audio analysis results, Gemini evaluations, production quality checks, and channel-oriented metrics.

### Settings

Application preferences, paths, render presets, API keys, FFmpeg configuration, and workflow options.
