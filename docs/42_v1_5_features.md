# v1.5 Features

Creator OS v1.5 expands beyond the v1.0 vocal workflow into relaxing production, shorts production, visual consistency, and review operations.

## Features

1. Relaxing production window
2. Room Profile
3. Journey Profile
4. Ambient Scanner
5. Ambient Mixer
6. Target-duration loop render
7. Shorts Studio
8. Hook Finder
9. Player UI Theme
10. Visual Bible
11. Human Review Queue
12. Project Archive

## v1.5 Scope Rules

- v1.5 builds on the v1.0 project, render, logging, and settings foundation.
- Relaxing production should support Somnera and Noctis Atlas workflows.
- Shorts Studio should create derived content from long-form videos.
- Visual Bible should support vocal album image consistency.
- Human Review Queue should collect low-confidence or approval-sensitive items.
- Project Archive should preserve completed or inactive projects without losing traceability.
