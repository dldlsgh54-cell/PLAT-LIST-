# English Lyrics Checker

Vocal project lyrics are based on English.

The English Lyrics Checker validates lyrics before Suno production and subtitle generation.

## Checks

- Grammar check
- Awkward expression check
- Hard-to-pronounce sentence check
- Excessive chorus repetition check
- AI-like expression check
- Subtitle line length check

## Subtitle Rules

- Recommended line length: 6-10 words per line
- Automatically wrap overly long sentences
- Apply mobile safe area rules

## Purpose

- Improve lyric quality before music generation.
- Reduce awkward vocal phrasing.
- Avoid subtitles that are too long on screen.
- Keep lyrics natural and singable.
- Catch repetitive or generic AI-style writing before production.

## Workflow Position

1. Import lyrics from Excel.
2. Run English Lyrics Checker.
3. Show issues and recommendations.
4. User approves or edits lyrics.
5. Generate Suno work list.
6. Use approved lyrics as the subtitle source of truth.
