# AI Decision System

Gemini does not make final decisions alone.

Creator OS combines Python analysis, Gemini evaluation, and Creator OS rules to produce a final recommendation.

## Final Decision Weight

- Python analysis: 40%
- Gemini evaluation: 40%
- Creator OS rules: 20%

## Python Analysis Responsibilities

- Length
- BPM
- Key
- LUFS
- Peak
- Clipping
- Silence
- RMS
- Dynamic range
- MFCC
- Chroma
- Audio fingerprint
- Internal similarity

## Gemini Evaluation Responsibilities

- Emotional delivery
- Vocal naturalness
- Chorus strength
- Immersion
- Commercial potential
- Replay value
- Album flow
- Brand fit

## Creator OS Rule Responsibilities

- 60-minute requirement
- Brand rules
- Duplicate prevention
- Subtitle existence
- Image existence
- Render readiness

## Decision Rules

- Gemini output is treated as evaluation data, not final approval.
- Creator OS should show the reason behind each recommendation.
- Final approval belongs to the user.
- Rule failures can block rendering even when Python and Gemini scores are high.
- Python analysis should provide measurable signals.
- Gemini evaluation should provide qualitative signals.
