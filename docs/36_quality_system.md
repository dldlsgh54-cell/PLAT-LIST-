# Quality System

Creator OS uses the Creator Quality Index, or CQI.

CQI summarizes the quality and release readiness of a project or asset.

## CQI Items

- Music Quality
- Vocal Quality
- Mix Quality
- Album Flow
- Diversity
- Originality
- Brand Fit
- Subtitle Accuracy
- Visual Match
- Length Requirement

## Score Criteria

- 90 or higher: Ready for release
- 80-89: Needs review
- Under 80: Recommend regeneration

## Rules

- CQI should combine measurable analysis, Gemini evaluation, and Creator OS rule checks.
- CQI should show item-level scores, not only a final total.
- Length Requirement can block rendering even when the total CQI is high.
- Low CQI items should include clear reasons and suggested actions.
- User approval is still required before release.
