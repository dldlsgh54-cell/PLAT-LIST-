# Automation Rules

Creator OS applies automatic rules by channel.

## Somnera

- Type: Relaxing
- Profile: Room Profile
- Master duration: 50-70 minutes
- Render length: Loop to target duration
- Spectrum: Disabled
- Subtitles: Disabled

## Noctis Atlas

- Type: Relaxing
- Profile: Journey Profile
- Master duration: 50-70 minutes
- Render length: Loop to target duration
- Focus: Moving spaces

## Aurora Echo

- Type: Vocal
- Lyrics language: English
- Minimum duration: 60 minutes
- Subtitles: Enabled
- Spectrum: Enabled
- Focus: Female Pop

## Velvet Verse

- Type: Vocal
- Lyrics language: English
- Minimum duration: 60 minutes
- Subtitles: Enabled
- Spectrum: Enabled
- Focus: Cinematic Pop / Emotional Pop

## Rules

- Channel automation rules should apply by default when a project is created.
- Channel-specific hard rules can block render when violated.
- Relaxing channels use master playlist looping.
- Vocal channels require subtitles and spectrum.
- Somnera must not use spectrum or subtitles.
