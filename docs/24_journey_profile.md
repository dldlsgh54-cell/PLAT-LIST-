# Journey Profile

Noctis Atlas uses the Journey Profile system.

One video equals one journey.

## Journey Examples

### Midnight Train

- Train Interior
- Rail Rhythm
- Cabin Hum

### Ocean Liner

- Ocean Waves
- Ship Engine Hum
- Hull Creak

### Airship

- Gentle Engine
- Wind
- Cabin Hum

### Night Highway

- Engine Hum
- Tire Noise
- Distant Wind

## Ambient Recipe

Each Journey Profile has a fixed Ambient Recipe.

## Rules

- A Noctis Atlas video should stay inside one journey concept.
- Ambient assets must match the selected Journey Profile.
- Each journey defines its own required ambient categories.
- Ambient combinations should support the feeling of movement.
- Unapproved ambient categories should be blocked or flagged.
- Journey Profile selection controls the default ambient recommendations.
