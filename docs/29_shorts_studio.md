# Shorts Studio

Shorts Studio is a separate window.

Shorts are not new standalone content. They are Derived Content from long-form videos.

## Workflow

1. Select Long Video
2. Analyze YouTube data
3. Find popular sections or popular songs
4. Extract hook
5. Generate 20-second, 30-second, 45-second, and 60-second candidates
6. Run Gemini evaluation
7. Render vertical video
8. Connect long-form link
9. Upload

## Rules

- Every short must be linked to a source long-form video.
- Shorts should preserve source project, track, and playlist relationships.
- Candidate durations should include 20, 30, 45, and 60 seconds.
- Gemini evaluation is advisory, not final approval.
- Upload metadata should include or reference the related long-form video.
- Shorts production should support both vocal and relaxing long-form sources.
