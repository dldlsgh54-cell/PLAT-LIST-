# First Development Target

The first development target is the Vocal 60-Minute Playlist MVP.

This MVP establishes the core workflow that relaxing production and shorts production can later extend.

## MVP Order

1. Create project
2. Upload Excel lyrics
3. Import A/B audio
4. Automatically match A/B files
5. Analyze duration with Python
6. Run Gemini A/B evaluation
7. Select winner
8. Calculate total playlist duration
9. Block rendering under 60 minutes
10. Preview Render

## Core Outcome

When this MVP is complete, Creator OS should be able to:

- Create a vocal playlist project.
- Import lyrics and A/B audio files.
- Match A/B versions per track.
- Analyze track duration locally.
- Evaluate A/B versions with Gemini.
- Select winners.
- Calculate total selected playlist duration.
- Block render when the playlist is under 60 minutes.
- Generate a Preview Render.

## Expansion Rule

After this MVP is complete, Rest Studio and Shorts Studio should be built on the same core systems:

- Project model
- Asset import
- Internal IDs
- Non-destructive file handling
- Python analysis
- Gemini evaluation
- Preflight Check
- Render pipeline
- Logs
