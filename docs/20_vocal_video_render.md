# Vocal Video Render

Vocal videos are rendered as music player-style videos.

## Video Components

- Track-specific image or short loop video
- Player UI
- Audio spectrum
- Subtitles
- Song title
- Artist name
- Image transition crossfade

## Brand UI Presets

### Aurora Echo

- Colors: Purple, pink, white
- Spectrum style: Circular spectrum
- Mood: Dreamy glow

### Velvet Verse

- Colors: Black, gold, white
- Spectrum style: Minimal waveform
- Mood: Cinematic typography

## Render Rules

- Each track should display its assigned image or loop video.
- Track transitions should use image crossfade.
- Subtitles should respect mobile safe area rules.
- Player UI should remain readable across the full video.
- Brand UI presets should be reusable templates.
- Audio visualization should match the selected channel brand.
